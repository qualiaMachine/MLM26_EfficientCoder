**Sneak peek — BYOA-Coding 2026** (Bring Your Own Agent), one of this year's challenges I'm drafting. A Kaggle Community Hackathon where teams build the best coding agent that runs entirely on a single ≤48 GB GPU with open-weight models, scored on [Terminal-Bench](https://tbench.ai) — the same yardstick used to evaluate Claude Code, Codex, Cursor, etc. Two tracks: build the agent, or extend the eval methodology. Feeds directly into how we deploy AI for researchers on campus.
